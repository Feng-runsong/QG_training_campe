## 线性回归 vs 逻辑回归 异同点总结

| 方面             | 线性回归                           | 逻辑回归                                        |
| ---------------- | ---------------------------------- | ----------------------------------------------- |
| **输出**         | 连续值（如质量分数 3~8）           | 概率值，再转为 0/1 分类（好酒/坏酒）            |
| **假设函数**     | $( h_\theta(x) = \theta^T x )$     | $( h_\theta(x) = \frac{1}{1+e^{-\theta^T x}} )$ |
| **损失函数**     | 均方误差 (MSE)                     | 交叉熵损失 (Log Loss)                           |
| **求解方式**     | 有解析解（正规方程），也可梯度下降 | 只能迭代优化（如梯度下降）                      |
| **误差分布假设** | 目标变量服从正态分布               | 目标变量服从伯努利分布                          |
| **评价指标**     | MSE, RMSE, R²                      | 准确率、精确率、召回率、F1、混淆矩阵            |
| **适用场景**     | 数值预测（房价、温度、销量）       | 二分类问题（垃圾邮件、疾病诊断、好坏酒判别）    |

### 相同点
1. 都属于广义线性模型，使用特征的线性组合。
2. 都需要特征预处理（如标准化）以提升数值稳定性。
3. 模型可解释性强，权重反映特征对输出的影响方向。

### 不同点
1. **输出类型**：线性回归输出连续实数；逻辑回归输出 [0,1] 概率并用于分类。
2. **假设函数**：线性回归无激活函数；逻辑回归使用 sigmoid 函数将线性输出映射到概率空间。
3. **损失函数**：线性回归最小化 MSE；逻辑回归最小化交叉熵。
4. **求解方式**：线性回归有闭式解；逻辑回归通常只能通过迭代算法求解。
5. **误差分布**：线性回归基于高斯-马尔可夫假设；逻辑回归基于伯努利分布。
6. **评价指标**：回归任务使用误差和拟合优度；分类任务使用准确率、精确率、召回率等。
