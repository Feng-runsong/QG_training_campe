# 第一周思考题

## 1. 对于有向图，有权图，公式哪些参数需要改变？

经典的一致性算法（连续时间）为：

$$
\dot{x}_i(t) = \sum_{j \in N_i} a_{ij} \big( x_j(t) - x_i(t) \big)
$$

其中：
- $x_i$ 是智能体 $i$ 的状态
- $N_i$ 是智能体 $i$ 的邻居集合
- $a_{ij}$ 是边 $(j,i)$ 的权重（信息从 $j$ 流向 $i$）

### 对于有向图、有权图，需要改变的参数：

1. **邻接矩阵 $A$ 是非对称的**
   
- 有向图时，$a_{ij} \neq a_{ji}$ 允许存在，表示信息流向有方向性
   
2. **Laplacian 矩阵定义变化**
   - 无向图：$L = D - A$
   - 有向图：使用**入度矩阵** $D_{in}$
     $$
     L = D_{in} - A
     $$
     其中 $D_{in} = \text{diag}(d_1, \dots, d_n)$，$d_i = \sum_{j=1}^n a_{ij}$（行和）

3. **状态更新公式（离散时间）**
   - 离散时间一致性：
     $$
     x_i(k+1) = x_i(k) + \varepsilon \sum_{j=1}^n a_{ij} (x_j(k) - x_i(k))
     $$
   - 矩阵形式：$x(k+1) = (I - \varepsilon L) x(k)$

---

## 2. 推导公式的矩阵形式

### 连续时间情况

令 $x(t) = [x_1(t), \dots, x_n(t)]^T$，则：

$$
\dot{x}_i(t) = \sum_{j=1}^n a_{ij} (x_j - x_i) = \sum_{j=1}^n a_{ij} x_j - d_i x_i
$$

矩阵形式：

$$
\dot{x} = A x - D_{in} x = -(D_{in} - A) x = -L x
$$

其中 $L = D_{in} - A$ 是有向图的**入度 Laplacian**。

### 离散时间情况

$$
x(k+1) = x(k) - \varepsilon L x(k) = (I - \varepsilon L) x(k)
$$

其中 $\varepsilon$ 是步长，需满足收敛条件（通常 $0 < \varepsilon < \frac{2}{\max_i d_i}$）。

---

## 3. 智能体状态最终趋于一致的分析

### （1）一致值

对于**强连通且平衡**的有向图（或更一般地，**含有向生成树**且 **Laplacian 矩阵有简单零特征值**的情况），系统收敛到一致值：

**一致值公式**：

$$
x_{\text{final}} = \frac{\sum_{i=1}^n v_i x_i(0)}{\sum_{i=1}^n v_i}
$$

其中：
- $v = [v_1, \dots, v_n]^T$ 是 **$L$ 的左零特征向量**，满足 $v^T L = 0$
- $v_i \ge 0$，对应根节点的权重

**特殊情况**：
- **无向图/平衡有向图**：$v = \mathbf{1}$，一致值为初始状态的平均值
- **一般有向图**：一致值为初始状态的**加权平均**，权重由有向图结构决定

### （2）各种因素影响下的一致性

| 影响因素   | 是否还能趋于一致 | 条件/说明                                |
| ---------- | ---------------- | ---------------------------------------- |
| 切换拓扑   | ✅ 是             | 需满足**联合连通**条件，足够频繁的连通性 |
| 固定时延   | ✅ 是             | 时延在稳定性范围内（通常小于某个上界）   |
| 通信噪声   | ⚠️ 近似一致       | 状态围绕一致值波动，有界误差             |
| 时延过大   | ❌ 否             | 可能导致发散或振荡                       |
| 失去生成树 | ❌ 否             | 图不包含有向生成树时，无法全局一致       |

## 4. 离散时间和连续时间算法的异同

### 相同点

| 项目       | 说明                                                         |
| ---------- | ------------------------------------------------------------ |
| 核心思想   | 基于邻居信息调整状态，使状态差逐渐消失                       |
| 收敛条件   | 固定拓扑需包含有向生成树                                     |
| 最终一致值 | 由左特征向量决定的初始状态加权平均                           |
| 矩阵形式   | 连续：$\dot{x} = -L x$；离散：$x(k+1) = (I - \varepsilon L) x(k)$ |

### 不同点

| 对比项         | 连续时间                    | 离散时间                                                     |
| -------------- | --------------------------- | ------------------------------------------------------------ |
| **更新方式**   | 微分方程，持续更新          | 差分方程，离散时刻更新                                       |
| **收敛速度**   | 由 $L$ 的最小非零特征值决定 | 由 $P = I - \varepsilon L$ 的谱半径决定，依赖于 $\varepsilon$ |
| **步长约束**   | 无（物理系统有响应限制）    | 需满足 $0 < \varepsilon < \frac{2}{\max_i d_i}$（无向图）    |
| **实现方式**   | 模拟电路/连续控制           | 数字控制/周期性通信                                          |
| **时延敏感度** | 对固定时延更敏感            | 可显式建模时延为状态变量                                     |
| **通信要求**   | 需连续通信                  | 仅在离散时刻通信                                             |

---

## 总结

- 有向图、有权图下，核心改变在于 **Laplacian 矩阵采用入度定义**，最终一致值由**左特征向量**决定
- 连续时间和离散时间算法在数学本质上一致，但在**实现方式、收敛条件和步长选择**上存在差异
- 存在切换拓扑、适度时延、噪声等影响因素时，系统仍可能趋于一致，但需满足相应的稳定条件
